## Table Name: REPORTING_DEV.DBT_MAHMUDNABI_gtm_analytics.opportunity_product_snapshot

## Overview  
This query retrieves a complete snapshot of opportunity product details from the `sfdc_opportunity_product_snapshot` table, providing all available attributes for each opportunity line item.

- `TRANSFORM_DEV.DBT_MAHMUDNABI_snapshots.sfdc_opportunity_product_snapshot`: Snapshot of Salesforce opportunity product records.

## Logic Explanation  
1. **Source**: The query reads directly from the `sfdc_opportunity_product_snapshot` table in the `TRANSFORM_DEV.DBT_MAHMUDNABI_snapshots` schema.  
2. **Selection**: All columns listed in the SELECT clause are retrieved without any transformation or filtering.  
3. **No Joins or Aggregations**: The query contains no JOINs, WHERE clauses, GROUP BY, or ORDER BY statements.  
4. **Result**: Each row returned represents a single snapshot of an opportunity product, preserving all original attributes.

## Grain of Data  
The result set is at the **opportunity line item** level; each row corresponds to one opportunity product snapshot.
