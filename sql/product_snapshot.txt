## Table Name: REPORTING_DEV.DBT_MAHMUDNABI_gtm_analytics.product_snapshot

## Overview
A simple projection query that retrieves a wide set of product attributes from the Salesforce product snapshot table.

## Tables Involved

## Logic Explanation
The query performs a straightforward projection of selected columns from the single source table TRANSFORM_DEV.sfdc_product_snapshot. No joins, filters, aggregations, or derived calculations are present. The result set contains one row per record in the source table, with each column directly mapped to its corresponding field in the snapshot.

## Grain of Data
Row‑level granularity: each output row represents a distinct product snapshot record from the source table.
