## Table Name: REPORTING_DEV.DBT_MAHMUDNABI_gtm_analytics.quote_snapshot

## Overview  
This query retrieves a snapshot of Salesforce CPQ quote records, including metadata, billing, shipping, pricing, approval, and usage details, for downstream analytics or reporting.

## Tables Involved  
- **TRANSFORM_DEV.DBT_MAHMUDNABI_snapshots.sfdc_sbqq_quote_snapshot**: Snapshot of Salesforce CPQ quote data.

## Logic Explanation  
The query performs a straightforward projection of the listed columns from the `TRANSFORM_DEV.DBT_MAHMUDNABI_snapshots.sfdc_sbqq_quote_snapshot` table.  
- **Data source**: single table `sfdc_sbqq_quote_snapshot`.  
- **No joins**: the query references only this table.  
- **No filters**: all rows are returned.  
- **No aggregations or calculations**: all selected columns are directly taken from the source.  
- **Result set**: a one‑to‑one copy of the snapshot table rows with the specified columns.

## Grain of Data  
One row per quote snapshot record; each row represents a snapshot of a Salesforce CPQ quote at a specific point in time.
